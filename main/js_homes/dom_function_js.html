DOM & Function!

What is the DOM?

The DOM (Document Object Model) is a representation of an HTML document in a tree-like structure, where each element is a node. JavaScript can manipulate the DOM to dynamically change content, styles, and behavior of a webpage.

Example

document.getElementById("message").textContent = "Hello, World!";

Regular Function

A function is a reusable block of code that performs a specific task. Functions help organize and simplify your code.

Example

function greet(name) {

return "Hello, " + name + "!";

}

console.log(greet("Alice"));

Arrow Function

A function is a reusable block of code that performs a specific task. Functions help organize and simplify your code.

Example

const greet = name => `Hello, ${name}!`;

console.log(greet("Alice"));;

Basic Commands in JavaScript

JavaScript is a powerful and widely used programming language that allows developers to create interactive and dynamic web pages. It runs in web browsers and can be used to manipulate HTML and CSS, handle user interactions, and even develop complex web applications. Alongside HTML and CSS, JavaScript is one of the core technologies of the web. With JavaScript, developers can create everything from simple alerts to advanced applications like online games, interactive forms, and real-time updates without reloading the page. It is also widely used in backend development with technologies like Node.js.

What is JavaScript?

JavaScript provides several built-in commands that allow interaction with users and control over how the code executes:

alert("Message"): Displays a popup window with a message. Commonly used for simple notifications.

confirm("Are you sure?"): Opens a dialog box with "OK" and "Cancel" options, useful for confirming actions.

prompt("Enter your name"): Asks the user to input a value in a text box.

Control Structures

JavaScript allows decision-making and loops for repeated execution:

Conditional Statements (if / else): Used to execute different blocks of code based on conditions. Example: If a user is above 18, show "You are an adult"; otherwise, show "You are a minor".

Loops (for, while): Used to execute a block of code multiple times. Example: Display numbers from 1 to 10 automatically.

switch Statement: A more efficient way to handle multiple conditions instead of using multiple if / else statements.

Variables and Data Storage

In JavaScript, variables store data and can be declared using different keywords:

var: The traditional way to declare variables, but it has function scope, which may cause unexpected behavior.

let: A modern and recommended way to declare variables, limited to the block where it is defined.

const: Used to declare constants, meaning their value cannot be changed after assignment.
